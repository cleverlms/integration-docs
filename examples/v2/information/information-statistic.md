## Document statistic
### Response format
Report export in **JSON**

### Response structure
| Attribute | Description                                                                      | Data type     |
|-----------|----------------------------------------------------------------------------------|---------------|
| id        | Document ID                                                                      | int           |
| login     | Login                                                                            | string        |
| viewed_at | Document view date. Format is Y-m-d\TH:i:sP (example, 2020-12-01T15:52:01+03:00) | string / null |
